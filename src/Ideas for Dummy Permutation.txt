Algorithm with Deterministic Permutations
Key Insight: Seed-Based Reproducibility
Each experiment k gets its own RNG seeded with seed_ + k, so we can recreate the exact same permutation in any T_stop iteration:

cpp
// Experiment k at T_stop = 1:
std::mt19937 rng_k(seed_ + k);
auto D_k = applyPermutation(base_dummies_, rng_k);

// Later, experiment k at T_stop = 2:
std::mt19937 rng_k(seed_ + k);  // ✅ Same seed!
auto D_k = applyPermutation(base_dummies_, rng_k);  // ✅ Same permutation!


Eigen::MatrixXd TRexSelector::applyRowAndColumnPermutation(
    const Eigen::MatrixXd& dummies,
    std::mt19937& rng
) const {
    /*
        Apply row and column permutation to dummy matrix.

        Row permutation: Shuffles elements within each column
        Column permutation: Shuffles which column is which

        This creates genuinely different dummy variables.
    */

    const std::size_t n = dummies.rows();
    const std::size_t L = dummies.cols();

    // Step 1: Row permutation (shuffles elements)
    std::vector<std::size_t> row_indices(n);
    std::iota(row_indices.begin(), row_indices.end(), 0);
    std::shuffle(row_indices.begin(), row_indices.end(), rng);

    Eigen::MatrixXd row_permuted(n, L);
    for (std::size_t i = 0; i < n; ++i) {
        row_permuted.row(i) = dummies.row(row_indices[i]);
    }

    // Step 2: Column permutation (shuffles dummy order)
    std::vector<std::size_t> col_indices(L);
    std::iota(col_indices.begin(), col_indices.end(), 0);
    std::shuffle(col_indices.begin(), col_indices.end(), rng);

    Eigen::MatrixXd permuted(n, L);
    for (std::size_t j = 0; j < L; ++j) {
        permuted.col(j) = row_permuted.col(col_indices[j]);
    }

    return permuted;
}


Get Dummies for Experiment k
cpp
Eigen::MatrixXd TRexSelector::getDummiesForExperiment(
    std::size_t experiment_index
) {
    /*
        Retrieve (or generate) dummy matrix for experiment k.

        Strategy: Dummy Permutation
        - Generate base_dummies_ once
        - For experiment k: apply deterministic permutation based on (seed_ + k)
        - Same permutation is recreated across T_stop iterations
    */

    const std::size_t L = trex_ctrl_.max_num_dummies;

    if (trex_ctrl_.use_dummy_permutation) {

        // First call: generate base dummies (once)
        if (!base_dummies_initialized_) {
            if (verbose_) {
                printProgress("Generating base dummy matrix (n=" +
                             std::to_string(n_) + ", L=" + std::to_string(L) + ")");
            }

            // Use main seed for base generation
            std::mt19937 base_rng(seed_);
            base_dummies_ = generateStandardDummies(L, base_rng);
            base_dummies_initialized_ = true;
        }

        // Create experiment-specific RNG (deterministic)
        std::mt19937 experiment_rng(seed_ + experiment_index + 1);

        // For experiment 0, skip permutation (use base directly)
        if (experiment_index == 0) {
            return base_dummies_;
        }

        // For experiments k > 0, apply deterministic permutation
        return applyRowAndColumnPermutation(base_dummies_, experiment_rng);

    } else {
        // Standard: generate fresh dummies for each experiment
        std::mt19937 rng(seed_ + experiment_index + 1);
        return generateStandardDummies(L, rng);
    }
}

Statistical Independence
For experiment k at any T_stop iteration:

cpp
std::mt19937 rng_k(seed_ + k + 1);  // Always the same seed for experiment k

Result:

✅ Experiment k always gets the same permuted dummies across T_stop iterations

✅ Different experiments get different permuted dummies

✅ Row permutation ensures dummy vectors are genuinely different

✅ Reproducible with same global seed

Augmented Matrix Structure
text
Experiment 0: X_aug = [X | D]                    (base)
Experiment 1: X_aug = [X | P_1(D)]               (permuted)
Experiment 2: X_aug = [X | P_2(D)]               (different permutation)
...
Experiment K-1: X_aug = [X | P_{K-1}(D)]
Where P_k(D) is a deterministic row+column permutation based on seed seed_ + k + 1.

Key Point: Restoration is Automatic
You asked about "difficulty to restore the correct D for each random experiment."

Solution: We don't store the permuted dummies! We regenerate them on-demand using the deterministic RNG:

text
T_stop = 1:
  Experiment k: rng_k(seed_ + k) → P_k(D) → run selection → discard P_k(D)

T_stop = 2:
  Experiment k: rng_k(seed_ + k) → P_k(D) → run selection → discard P_k(D)
                ^^^^^^^^^^^^^^^ Same seed! Same permutation!
Memory: Only store base_dummies_ (n × L), not K permuted versions!



Question: Does row permutation preserve L2 norm?
Answer: YES! ✅

text
Original column: d = [d₁, d₂, ..., dₙ]ᵀ
||d||₂ = √(d₁² + d₂² + ... + dₙ²)

After row permutation: d' = [d_π(1), d_π(2), ..., d_π(n)]ᵀ
||d'||₂ = √(d_π(1)² + d_π(2)² + ... + d_π(n)²) = ||d||₂
Norm is invariant under permutation! 🎉

Optimized Implementation
cpp
Eigen::MatrixXd TRexSelector::getDummiesForExperiment(
    std::size_t experiment_index
) {
    const std::size_t L = trex_ctrl_.max_num_dummies;

    switch (trex_ctrl_.dummy_strategy) {

        case DummyStrategy::PERMUTATION_ROW:
        case DummyStrategy::PERMUTATION_BOTH: {

            // Initialize and normalize base dummies once
            if (!base_dummies_initialized_) {
                std::mt19937 base_rng(seed_);
                base_dummies_ = generateStandardDummies(L, base_rng);

                // ✅ L2 normalize base dummies ONCE
                for (std::size_t j = 0; j < L; ++j) {
                    base_dummies_.col(j) /= base_dummies_.col(j).norm();
                }

                base_dummies_initialized_ = true;
            }

            // Experiment 0: use base directly
            if (experiment_index == 0) {
                return base_dummies_;
            }

            // Experiments k > 0: permute (no re-normalization needed!)
            std::mt19937 exp_rng(seed_ + experiment_index + 1);

            if (trex_ctrl_.dummy_strategy == DummyStrategy::PERMUTATION_ROW) {
                return applyRowPermutation(base_dummies_, exp_rng);
                // ✅ No normalization needed - permutation preserves norm!
            } else {
                return applyRowAndColumnPermutation(base_dummies_, exp_rng);
                // ✅ No normalization needed - permutation preserves norm!
            }
        }

        case DummyStrategy::STANDARD:
        default: {
            std::mt19937 exp_rng(seed_ + experiment_index + 1);
            Eigen::MatrixXd dummies = generateStandardDummies(L, exp_rng);

            // Normalize fresh dummies
            for (std::size_t j = 0; j < L; ++j) {
                dummies.col(j) /= dummies.col(j).norm();
            }

            return dummies;
        }
    }
}

Norm Preservation Analysis
Row Permutation (Applied to Entire Matrix)
cpp
// Original normalized matrix:
D = [d₁ | d₂ | ... | d_L]  where ||d_j||₂ = 1 for all j

// Apply row permutation σ:
// For each column j, elements [D[1,j], D[2,j], ..., D[n,j]]
// become [D[σ(1),j], D[σ(2),j], ..., D[σ(n),j]]

D'[i,j] = D[σ(i), j]

// Norm of column j after row permutation:
||d'_j||₂ = sqrt(D[σ(1),j]² + D[σ(2),j]² + ... + D[σ(n),j]²)
          = sqrt(D[1,j]² + D[2,j]² + ... + D[n,j]²)  // same terms, different order
          = ||d_j||₂ = 1  ✅
Row permutation preserves column norms! ✅

Column Permutation
cpp
// Start with D' (after row permutation, columns still normalized)
D' = [d'₁ | d'₂ | ... | d'_L]  where ||d'_j||₂ = 1

// Apply column permutation π:
D'' = [d'_π(1) | d'_π(2) | ... | d'_π(L)]

// Each column d'_π(j) is just a reordering of which column is where
// The column itself is unchanged!
||d''_k||₂ = ||d'_π(k)||₂ = 1  ✅
Column permutation preserves norms trivially - we're just reordering columns! ✅

Combined: Row + Column Permutation
cpp
Eigen::MatrixXd TRexSelector::applyRowAndColumnPermutation(
    const Eigen::MatrixXd& dummies,
    std::mt19937& rng
) const {
    /*
        Combined row + column permutation.

        Both operations preserve L2 norms independently!
        No re-normalization needed.
    */

    const std::size_t n = dummies.rows();
    const std::size_t L = dummies.cols();

    // Step 1: Row permutation (preserves column norms)
    std::vector<std::size_t> row_indices(n);
    std::iota(row_indices.begin(), row_indices.end(), 0);
    std::shuffle(row_indices.begin(), row_indices.end(), rng);

    Eigen::MatrixXd row_permuted(n, L);
    for (std::size_t i = 0; i < n; ++i) {
        row_permuted.row(i) = dummies.row(row_indices[i]);
    }
    // ✅ All columns still have norm 1

    // Step 2: Column permutation (just reorders columns)
    std::vector<std::size_t> col_indices(L);
    std::iota(col_indices.begin(), col_indices.end(), 0);
    std::shuffle(col_indices.begin(), col_indices.end(), rng);

    Eigen::MatrixXd permuted(n, L);
    for (std::size_t j = 0; j < L; ++j) {
        permuted.col(j) = row_permuted.col(col_indices[j]);
    }
    // ✅ All columns STILL have norm 1

    // ✅ NO re-normalization needed!
    return permuted;
}
